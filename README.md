# PickBazaar

Full dropshipping shop with Snipcart automatic payments.

## Files
- index.html — Customer shop (homepage)
- my-shop.html — Your admin dashboard
- supplier-guide.html — How to connect suppliers

## Deploy to Netlify
Drag this folder to app.netlify.com — live in 60 seconds.

## Activate Payments (Snipcart)
1. Go to snipcart.com → sign up free
2. Dashboard → API Keys → copy your Public API Key
3. In index.html → search "YOUR_SNIPCART_PUBLIC_KEY" → replace with your key
4. Connect Stripe inside Snipcart dashboard → Payment gateway → Stripe
5. Done — every product price is read automatically, no manual links needed
